# News-website-front-end
This project is a responsive front-end news website inspired by popular portals like Prothom Alo. It is built using pure front-end technologies with a focus on clean UI, usability, and strong HTML/CSS/JavaScript fundamentals.
Navbar with Home, National, International, Sports, technology and Business
Home page with featured news, category filtering, search, and news grid.

**Technologies Used**

1. HTML5 – Semantic markup and structure

2. Tailwind CSS – Utility-first styling for rapid UI development

3. DaisyUI – Prebuilt Tailwind components for consistency and speed

4. Vanilla JavaScript – DOM manipulation, filtering, search, and routing logic

**Although no frameworks are used, the project follows a component mindset:**

1. Navbar

2. Featured section

3. News cards

4. Details view

5. Each section has a clear responsibility, improving readability and scalability.

**Features Implemented**

Responsive Navbar (Desktop & Mobile)

Featured News Section

News Grid with Cards

Category Filtering

Search Functionality

Skeleton Loading State

News Details Page

Smooth Hover & Transition Effects
